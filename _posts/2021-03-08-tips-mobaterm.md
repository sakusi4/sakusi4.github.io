MobaXTerm과 Windows Terminal입니다.

MovaXTerm
---
![0](/assets/images/2021-03-08-tips-mobaterm/mobaterm.png)

추천 대상
- CUI 환경(터미널)에 익숙하지 않다.
- 드래그 앤 드롭으로 편하게 파일을 업로드하고 다운로드 하고싶다.
- GUI 프로그램을 실행해야한다. (자체 X-Server가 있으므로 별도의 설정 불필요)
- 파일 탐색기에서 바로 텍스트 파일을 열 수 있다. (내장 편집기가 꽤 깔끔합니다.)

만능 클라이언트입니다. 단점이라면 불필요한 메뉴들이 좀 많습니다. 하지만 옵션 설정으로 보이지 않게 할 수 있습니다. 리눅스에 익숙하지 않은 윈도우 유저들에게 아주 추천합니다.

Windows Terminal
---
![0](/assets/images/2021-03-08-tips-mobaterm/powerline-powershell.png)

참고 자료: [Powerline 적용 방법][1]

[1]: https://docs.microsoft.com/en-us/windows/terminal/tutorials/powerline-setup

추천 대상
- 터미널 환경에 익숙하다.
- 깔끔한 폰트와 뚜렷한 색감
- 최소한의 기능

이 프로그램은 Microsoft Store에서 받을 수 있습니다. 우분투 데스크탑을 오래 사용한 저에게 윈도우의 커맨드창은 부족한 점이 많습니다. Windows Terminal은 zsh같은 터미널에 밀리지 않습니다. cmd, powershell, git bash 대신에 만족스럽게 사용중입니다.
