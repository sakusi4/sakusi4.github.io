포인터(Pointer)
---
포인터는 메모리 주소를 저장하는 변수다. 포인터를 어렵다고 여기는 사람들이 종종 있는데, 책에서 포인터 이전의 내용을 겉핥기로 배웠기 때문이다.

요컨대 변수를 공부하면서 상자에 값을 집어넣는다는 정도의 수준으로 공부했다면 포인터를 접했을 때 어렵게 느낄 수 있다. 글이 좀 길지만, 차분히 읽어보길 바란다.

포인터는 왜 필요할까?
---

{% highlight c linenos=table %}
// Example 1
void increase(int b)
{
    b++;
}

int main()
{        
    int a = 0;
    increase(a);
}
{% endhighlight %}
